# Add Package to DESCRIPTION

Adds a package to the Imports field in DESCRIPTION.

## Usage

``` r
add_to_description(package, field = "Imports", path = ".")
```

## Arguments

- package:

  Character. Package name.

- field:

  Character. DESCRIPTION field. Default: "Imports".

- path:

  Character. Path to project root.

## Value

Logical indicating success
